;
; Argument out of range exception compound message.  Arguments are:
; 0: The name of the argument that was found to be invalid.
; 1: The minimum allowed value of this argument.
; 2: The maximum allowed value of this argument.
; 3: The actual, invalid value of this argument.
;
argumentOutOfRange = Argument {0} outside of range [{1}, {2}]: {3}
;
; Argument type exception compound message.  Arguments are:
; 0: The name of the argument that was found to be invalid.
; 1: The full name of the argument's declared type.
; 2: The full name of the argument's required type.
; 3: The full name of the argument's actual type.
;
argumentType = Argument {0}, declared as {1}, must be castable to {2}, but supplied type was {3}
;
; Counter decrement overflow message.
;
counterDecrementOverflow = Attempt to decrement Counter below zero failed.
;
; Counter increment overflow message.  The argument is the counter's limit.
;
counterIncrementOverflow = Attempt to increment Counter (limit {0}) failed.
;
; Invalid state transition.  Arguments are:
; 0: The full name of the associated state context object's type.
; 1: The full name of the associated state context object's current state's
;    type.
; 2: The full name of the associated state context object's forbidden new
;    state's type.
;
invalidStateTransition = Attempt to modify state context object {0}'s state from {1} to {2}, which is forbidden
;
; Measure overflow message.  Arguments are:
; 0: The full name of the associated measurement unit type.
; 1: The minimum allowed standard unit value for this type of measure.
; 2: The maximum allowed standard unit value for this type of measure.
; 3: The actual, invalid value of this argument.
;
measureOverflow = An operation has caused a Facsimile.Common.Measure <{0}> value to be outside its valid range [{1}, {2}]: {3}
;
; Invalid state transition.  Arguments are:
; 0: The full name of the associated state context object's type.
; 1: The full name of the associated state context object's current state's
;    type.
; 2: The full name of the associated state context object's forbidden new
;    state's type.
;
stateChangeInProgress = Attempt to modify state context object {0}'s state to {1} failed because it has not yet completed a previous state change.
;
; Test messages.  These messages must not be translated and are required for
; successful execution of the test suite.  They should not be referenced except
; from within the test suite.
;
testMessage = Test message.
testCompoundMessage = Test compound message: {0}.
